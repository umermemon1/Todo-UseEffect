# ✅ Todo App (React + useEffect)

A simple and responsive **Todo Application** built using React.js, showcasing state management and side effects using the `useEffect` hook. The app allows users to add, delete, and manage tasks efficiently.

## 🛠 Tech Stack

- ⚛️ React.js – Frontend library for building UI  
- 🎣 useEffect – React hook for handling side effects  
- 🧠 useState – State management  
- 🎨 CSS – Styling  

---

## ✨ Features

- 📝 Add new todos  
- ❌ Delete todos  
- ✔️ Mark tasks as completed (if implemented)  
- 💾 Persistent UI updates using `useEffect`  
- 📱 Responsive design  
- ⚡ Real-time state synchronization  

---

## ⚙️ How useEffect is Used

This project uses `useEffect` to handle side effects such as:

- Syncing todos with local storage (if implemented)
- Running code when the component mounts
- Updating UI when todo state changes
